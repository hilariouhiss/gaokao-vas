# 关于

## 版本
1. 项目框架: `Spring Boot:3.2.1`
2. 数据库: `MySQL:8`, 使用 `Spring Data JPA` 操作数据库
3. 依赖管理: `Maven:3.9.6`
4. 项目启动端口: `8080`
5. JDK 版本: `JDK21`
6. 使用 Docker 部署

## 使用方法
1. 部署: 在项目根目录运行 `docker-compose up -d` 即可, 等待一段时间即可完成部署
2. 停止: 在项目根目录运行 `docker-compose down` 即可
